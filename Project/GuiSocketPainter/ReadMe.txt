
1.项目说明：此项目使用IDEA编辑器创建，但是在Eclipse中也能正常运行。

2.项目中对于图形的reshape操作， 对于矩形，只能通过移动左上角和右下角的点来说实现对矩阵的reshape操作
  对于正方形，只能操作左下角的点来时间reshape操作，也就是说，如果你想reshape某个图形，必须将鼠标移动
  到上面说的那些点。

4.此次项目让我更加深刻的理解了内部类的概念，线程的管控，Socket通信，JFrame图形界面。
